About This Project

This application is the persistent database layer for the multiplayer quiz game from Lab 10. It uses JPA (Java Persistence API) with Hibernate as the provider and Spring Data JPA for repositories. All quiz data (players, questions, games, results) is stored in an embedded H2 database.

Spring Boot 3.2.5 Hibernate 6.4 Spring Data JPA H2 Database Java 17 Criteria API JPQL

Features Implemented

📋

5 JPA Entities

Player, Question, Game, GameResult, AuditLog with proper relationships

🔗

Relationships

One-to-Many and Many-to-Many with bidirectional mapping

🔎

JPQL Queries

Custom read and transactional modifying queries with timing logs

🔧

Criteria API

Dynamic search with multiple optional filters on game results

📒

Auditing

Automatic audit trail for all database changes

🏆

REST API

Full CRUD endpoints for all entities
